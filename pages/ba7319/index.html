日志

# 日志

# 查看

$ git log

# 简化版日志

$ git log --pretty=oneline

# 查看前N条

git log -n

# 变更日志

-n 同上,不加则显示全部

git log --stat -n

# 查看提交修改

查看某次commit做了哪些修改

git show <commit-hash-id>

# 退出log状态

有时候日志过长，可以按下英文q退出状态

# 查看HEAD指向历史和说明

$ git reflog
